Mengenal Pikul

PIKUL merupakan singkatan dari Pinjaman Kuliah. Pikul menyajikan sistem peringanan beban mahasiswa yang kurang mampu berupa kredit barang yang dibutuhkan untuk perkuliahan seperti laptop, buku dan sebagainya.

Pikul dibuat menggunakan teknologi terbaru yaitu:

  • Electron.js 11.1.1
  • Node.js 12.18.4
  • MySQL 2.18.1

Pikul dibuat oleh Awal Ariansyah, sebagai pemenuhan Tugas Basis Data Lanjutan jurusan Teknik Informatika Semester 3 STMIK Komputama Majenang.

Source code aplikasi tersedia di link berikut bagi yang ingin mempelajari: https://github.com/awalariansyah/pikul
