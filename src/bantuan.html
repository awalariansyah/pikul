Bantuan Penggunaan Aplikasi


Jendela Pinjaman Kuliah

Jendela ini berisikan data mahasiswa yang melakukan transaksi pinjaman.

  • Pikul Digunakan untuk membuat pinjaman baru mahasiswa.
  • Cicil Digunakan untuk membuat cicilan bagi mahasiswa yang memiliki tagihan.
  • Telat Digunakan untuk menandai mahasiswa yang terlambat membayarkan tagihan.

Lainnya:

  • Apabila mahasiswa telah melunasi tagihan maka otomatis akan hilang dari tabel daftar pikulan.
  • Untuk memasukan data otomatis ke dalam form, klik langsung nama yang ingin dimasukkan pada daftar tabel.
  • Jumlah cicilan tidak boleh melebihi nilai tagihan.
Jendela Mahasiswa

Jendela ini berisikan data mahasiswa yang terdaftar dalam aplikasi.

  • Tambah Digunakan untuk menambahkan mahasiswa baru.
  • Ubah Digunakan untuk mengubah data mahasiswa.
  • Hapus Digunakan untuk menghapus data mahasiswa.

Lainnya:

  • Untuk menambahkan data mahasiswa baru, cukup isikan NIM dan Nama pada baris paling atas di tabel dan klik tombol Tambah.
  • Untuk mengubah data mahasiswa, klik baris mahasiswa yang ingin diubah lalu ubah data langsung pada sel tersebut selanjutnya klik tombol Ubah untuk menyimpan perubahan.
  • Untuk menghapus data mahasiswa, urutkan baris data yang akan diubah lalu klik tombol Hapus di sebelah kanan.
Jendela Barang

Jendela ini berisikan data barang yang telah dijadikan transaksi. Sebelum seorang mahasiswa melakukan peminjaman, maka barang harus di inputkan terlebih dahulu di jendela barang apabila barang tidak ada dalam daftar.

  • Tambah Digunakan untuk menambahkan barang baru.
  • Ubah Digunakan untuk mengubah data barang.
  • Hapus Digunakan untuk menghapus data barang.

Lainnya:

  • Untuk menambahkan data barang baru, cukup isikan Nama dan Harga pada baris paling atas di tabel dan klik tombol Tambah.
  • Untuk mengubah data barang, klik baris mahasiswa yang ingin diubah lalu ubah data langsung pada sel tersebut selanjutnya klik tombol Ubah untuk menyimpan perubahan.
  • Untuk menghapus data barang, urutkan baris data yang akan diubah lalu klik tombol Hapus di sebelah kanan.
Jendela Bantuan

Jendela Tentang
